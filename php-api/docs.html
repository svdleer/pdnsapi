Overview

PHP API wrapper for PowerDNS Admin with local database storage for extended functionality.

Key Features

  • Account Management - CRUD operations with local IP storage
  • Domain Management - Create/update domains via PowerDNS Admin API
  • Domain-Account Links - Associate domains with accounts
  • IP Allowlist - Global API access control
  • Templates - Reusable domain configurations
  • Data Sync - PowerDNS Admin integration

Technical Notes

  • Authentication: X-API-Key header required
  • IP Security: Global allowlist applies to all endpoints
  • Local Storage: IP addresses stored locally only
  • Safety: Domain deletion disabled for data protection

Quick Endpoint Reference

GET /accounts
Get all accounts with IP addresses
POST /accounts
Create account with optional IP addresses
PUT /accounts?id={id}
Update account including IP addresses
GET /domains
Get all domains
GET /domains?sync=true
Sync domains from PDNSAdmin
POST /domains
Create domain (auto-assigns to account)
PUT /domains?id={id}
Update domain (syncs account in PDNSAdmin)
GET /status
API status and health check
