Overview

The PDNSAdmin PHP API is a comprehensive wrapper for PowerDNS Admin that provides local database storage and extended functionality for managing DNS accounts and domains.

Key Features

🏢 Account Management

Create, read, update, and delete accounts with IP address storage (local only)

🌐 Domain Management

Manage domains with automatic PDNSAdmin synchronization (creation and updates only)

🔗 Domain-Account Links

Associate domains with accounts, automatically updating both local DB and PDNSAdmin

📡 Data Synchronization

Sync domains and accounts from PDNSAdmin API for offline access

🗄️ Local Database

Store data locally for better performance and extended functionality

🔒 Safety Features

Domain deletion disabled, comprehensive validation, and error handling

Important Notes

  • IP Addresses: Stored locally only, not sent to PDNSAdmin
  • Domain Deletion: Disabled for safety - domains can only be created and updated
  • Account Sync: Account changes automatically sync with PDNSAdmin
  • Domain Assignment: Domain-account assignments update both local DB and PDNSAdmin
